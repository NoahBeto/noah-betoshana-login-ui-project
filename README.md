# Login UI Project

This project is a simple login user interface built using HTML and CSS. It was created as the final project for Module 1 at Bay Valley Tech Code Academy. The goal of the project was to demonstrate an understanding of basic page structure, styling, and layout using core web technologies.

## Project Files

1. index.html  
   Contains the structure of the login page.

2. style.css  
   Contains all styling for the login interface.

## How to View the Project

1. Download or clone the repository.
2. Open the index.html file in any modern web browser.

No additional setup or dependencies are required.

## What This Project Includes

1. A clean and simple login form layout
2. Styling created entirely with CSS
3. A static frontend interface with no backend or authentication logic

## Purpose

This project was built for educational purposes as part of the Bay Valley Tech Code Academy curriculum. It focuses on practicing HTML structure and CSS styling rather than functionality or data handling.
